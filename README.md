# Python视觉开发环境搭建与图像基本读写实验

## 一、项目概述
本项目是计算机视觉课程的基础实验，主要完成Python视觉开发环境的搭建，以及图像的读取、显示、信息查看、格式转换、结果保存、像素访问与图像裁剪等核心操作。通过本次实验，掌握OpenCV、NumPy、Matplotlib三大库的基本使用方法，理解数字图像的存储原理，熟悉Git项目提交规范，为后续更复杂的图像处理任务打下基础。

## 二、实验环境与技术栈
1. 开发语言：Python 3.x
2. 图像处理库：OpenCV-Python
3. 数值计算库：NumPy
4. 绘图显示库：Matplotlib
5. 版本控制：Git / GitLab

## 三、项目文件说明
本项目包含以下文件：
- demo.py：实验主程序，实现所有图像操作功能
- test.jpg：实验使用的原始测试图片
- gray_test.jpg：程序运行后生成的灰度图像
- crop_test.jpg：程序运行后生成的裁剪图像
- README.md：项目说明文档

## 四、环境配置方法
在使用本项目前，需要安装相关依赖库，打开命令提示符或终端，执行以下命令：
pip install opencv-python numpy matplotlib

安装完成后，可以运行简单代码验证环境是否正常：
import cv2
import numpy
import matplotlib
print(cv2.__version__)

如果能够正常输出版本号，说明环境配置成功。

## 五、项目运行步骤
1. 将demo.py与test.jpg放在同一个文件夹中
2. 打开终端，进入项目所在目录
3. 运行命令：python demo.py
4. 程序会自动输出图像信息并弹出图像显示窗口
5. 运行结束后，文件夹中会自动生成处理后的灰度图和裁剪图

## 六、核心功能详细解析
### 1. 图像读取功能
使用OpenCV的imread函数读取本地图片，同时判断图片是否读取成功，避免因路径错误导致程序崩溃。

### 2. 图像基本信息输出
通过图像的shape属性获取图像的高度、宽度和通道数，通过dtype获取数据类型，在控制台打印输出，让我们了解图像的基本结构。

### 3. 彩色图像显示
OpenCV默认读取的图像为BGR格式，而Matplotlib需要RGB格式才能正常显示，因此需要进行颜色空间转换，转换后再显示图像。

### 4. 灰度图像转换与显示
将彩色三通道图像转换为单通道灰度图像，转换后使用Matplotlib显示，并设置灰度配色方案，保证显示效果正常。

### 5. 图像保存功能
将处理完成的灰度图像保存到本地，方便后续查看与使用，使用OpenCV的imwrite函数实现保存。

### 6. 像素值获取与图像裁剪
通过NumPy数组索引获取指定位置的像素值，了解图像在计算机中的存储方式；使用数组切片实现图像局部区域裁剪，并保存裁剪结果。

## 七、实验中遇到的问题及解决方法
1. 图像显示颜色异常
原因：OpenCV与Matplotlib颜色格式不同
解决：将BGR格式转换为RGB格式再显示

2. 无法读取图片
原因：图片路径错误或文件名错误
解决：检查路径，确保代码与图片在同一目录

3. Git仓库首页不显示说明文档
原因：README文件命名错误、有空格或后缀不正确
解决：严格命名为README.md，无空格、无多余后缀

4. 图像裁剪位置不正确
原因：坐标顺序混淆，OpenCV遵循先y后x的规则
解决：使用[y_start:y_end, x_start:x_end]格式进行切片
